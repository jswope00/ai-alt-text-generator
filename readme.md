# AI Powered Alt Text Generator

## Description

This [Streamlit](https://streamlit.io) micro-app creates alt text for accessibility from images. 

For most images, it provides brief alt text to describe the image, focusing on the most important information first. 

For complex images, like charts and graphs, the app creates a short description of the image and a longer detail that describes what the complex image is conveying. 

For more information, see <a href="https://www.w3.org/WAI/tutorials/images/" target="_blank">W3C Images Accessibility Guidelines</a>

It work with either GPT-Vision.

## Accessing the App 
You can access the app on the Streamlit Cloud community at [alt-text.streamlit.app](https://mcq-wizard.streamlit.app/).

## Getting Started
To use all features of the app, you may need your own OpenAI API key. Don't have one yet? Create one on [the OpenAI website](https://platform.openai.com/account/api-keys). Once you have your API key, enter it into the app when prompted.
